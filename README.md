# ecommerce-vetronaviglio

Progetto derivato da `triathlon-starter`.

## Stack
- React 19 + TypeScript
- Vite
- TailwindCSS v4
- Supabase
- Vercel

## Quick Start
```bash
cd app
cp ../.env.example .env
npm install
npm run dev
```
