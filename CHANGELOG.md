# Changelog - Vetronaviglio E-commerce

## [0.2.0] - 2026-06-16
### Added
- Auth System: Login/Logout implementato con ruoli (Customer, Admin).
- Admin Dashboard: Pannello di controllo completo per gestione catalogo.
- Product Editor: Interfaccia avanzata con Tiptap (visual editing) e Image Uploader.
- Gestione Asset: Migrazione massiva immagini su Supabase Storage e normalizzazione URL.
- Catalogo Editoriale: Paginazione, filtri avanzati (Categoria, Capacità, Materiale) e griglia responsiva 1-2-4.
- Security: RLS abilitata su tutte le tabelle catalogo.
- Infrastructure: Configurazione iniziale Supabase + Vercel.
